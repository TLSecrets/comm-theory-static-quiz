# 📡 通信系统原理 · 刷题系统
> 期末复习专用 · 离线可用 · 数据本地存储 · 支持GitHub Pages部署
## 🚀 快速开始
### 方式一：本地打开（离线可用）
直接双击 `index.html` 即可在浏览器中打开使用，无需安装任何软件。
### 方式二：GitHub Pages 部署
1. 将整个仓库推送到 GitHub
2. 进入仓库 `Settings → Pages`
3. `Source` 选择 `main` 分支，`/(root)` 目录
4. 点击 `Save`，等待部署完成
5. 访问 `https://你的用户名.github.io/仓库名/`

## 🎯 功能特性
- **三种题型适配**：单选（点击即答）、判断（对/错按钮）、计算题（多行公式输入，关键词模糊匹配）
- **快捷键支持**：`1-4`单选答题、`Z/X`判断、`Enter`计算提交、`←→`切题、`Space`查看答案
- **深色模式**：完整适配夜间护眼，题图自动降亮度
- **进度可视化**：三色进度条（答对✅/答错❌/未答⬜），独立进度隔离
- **计时器**：单题精确定时，后台切页自动暂停，超时提醒
- **错题本**：多维筛选、排序、批量操作、两种重做模式
- **收藏系统**：自定义标签、标签筛选、批量管理
- **数据管理**：一键导出/导入JSON备份，支持换设备迁移
- **打印支持**：所有页面可打印，自动隐藏交互控件

## 🛠️ 题库维护
题库数据源位于 `js/question.js`，每道题的JSON结构：
```javascript
{
  id: 1,                    // 唯一ID
  type: 'single',            // 题型：single/judge/calc
  chapter: '第1-2章小测',    // 章节
  question: '从信号频谱的特点...',  // 题干
  options: [                 // 仅单选题有此字段
    { key: 'A', text: '周期的' },
    { key: 'B', text: '连续的' },
    { key: 'C', text: '离散的' },
    { key: 'D', text: '发散的' }
  ],
  answer: 'C',               // 标准答案
  explanation: '',           // 解析（可选）
  image: ''                  // 配图路径（可选，相对于images/目录）
}
```
添加新题时在 `questionList` 数组中新增对象即可，需清除localStorage中的 `csp_cache` 后刷新页面。
## 💾 数据存储说明
所有数据存储在浏览器localStorage中，分片隔离：
| Key | 内容 |
|-----|------|
| `csp_config` | 全局配置（主题、字体、计时、计分等） |
| `csp_records` | 答题记录（每题的答对/答错次数、历史） |
| `csp_records_archive` | 归档的旧答题记录（可恢复） |
| `csp_errors` | 错题ID列表 |
| `csp_collect` | 收藏题目ID + 自定义标签 |
| `csp_timer` | 总计时、单题用时、超时记录 |
| `csp_cache` | 题库缓存（首次加载后缓存，二次打开更快） |
